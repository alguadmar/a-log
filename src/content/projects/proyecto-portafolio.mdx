# Portafolio Personal

Este proyecto es un portafolio web minimalista y elegante construido con tecnologías modernas para mostrar mi trabajo y habilidades técnicas de manera efectiva.

## Características principales

- **Diseño responsivo**: Se adapta perfectamente a cualquier tamaño de pantalla, desde móviles hasta escritorio
- **Transiciones suaves**: Animaciones sutiles que mejoran la experiencia del usuario
- **Modo oscuro/claro**: Preferencia de tema con detección automática del sistema
- **Filtrado de proyectos**: Capacidad de filtrar proyectos por categoría o tecnología
- **Formulario de contacto**: Integración con servicios de correo electrónico sin servidor

## Tecnologías utilizadas

### Frontend
- Astro como framework principal
- Componentes React para interfaces interactivas
- TailwindCSS para estilos
- Framer Motion para animaciones

### Herramientas de desarrollo
- TypeScript para seguridad de tipos
- ESLint y Prettier para mantener código limpio
- GitHub Actions para CI/CD

## Proceso de desarrollo

El desarrollo del portafolio siguió un proceso estructurado:

1. Investigación y benchmarking de portafolios de referencia
2. Wireframing y diseño en Figma
3. Implementación de la estructura básica con Astro
4. Desarrollo de componentes interactivos con React
5. Estilización con TailwindCSS
6. Optimización de rendimiento y accesibilidad
7. Despliegue en Vercel

## Retos y soluciones

Durante el desarrollo, me enfrenté a varios desafíos:

- **Optimización de imágenes**: Utilicé el sistema de optimización de imágenes de Astro para cargar imágenes eficientemente
- **Transiciones entre páginas**: Implementé el sistema View Transitions API de Astro para transiciones suaves
- **Desempeño**: Prioricé el uso de componentes islandos para mantener un bundle mínimo

## Conclusiones y aprendizajes

Este proyecto me permitió profundizar en el ecosistema de Astro y mejorar mis habilidades en:

- Arquitectura de aplicaciones web modernas
- Diseño de experiencias de usuario intuitivas
- Optimización de rendimiento frontend
- Mejores prácticas de accesibilidad web

El código está disponible en GitHub y siempre estoy abierto a sugerencias y mejoras.
